# Faidx version 1.1

This is a Perl XS interface into the FAIDX component of htslib

## INSTALLATION

To install this module type the following:
```
perl Makefile.PL
make
make test
make install
```

## DEPENDENCIES

This module requires these other modules and libraries:

HTSLIB available from http://htslib.org
